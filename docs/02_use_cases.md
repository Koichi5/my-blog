# 👥 ユーザータイプとユースケース

## ■ ユーザータイプと権限

| ユーザー種別 | 権限概要 |
|---------------|----------|
| **管理者（Admin）** | 全ユーザー・投稿・コメント・いいねを管理可能 |
| **一般ユーザー（General）** | 記事投稿、コメント、いいねが可能 |
| **ゲスト（未ログイン）** | 記事閲覧、コメント投稿、いいね可能（Cookie識別） |

---

## ■ ユースケース一覧

| No | アクター | 機能概要 |
|----|-----------|----------|
| 1 | 一般ユーザー | 記事を投稿・編集・削除する |
| 2 | 一般ユーザー | Markdownで記事を作成する |
| 3 | 一般ユーザー | 他の記事を閲覧・コメント・いいねする |
| 4 | ゲスト | 記事を閲覧・コメント・いいねする（ログイン不要） |
| 5 | 管理者 | 不適切な記事・コメントを削除する |
