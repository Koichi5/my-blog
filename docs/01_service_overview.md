# 🏁 サービス概要

## ■ サービス名（仮）
MyBlogApp

## ■ 概要
個人が自分の考えや技術記事を発信できるブログ投稿サービス。
投稿された記事は一般ユーザーやゲストが閲覧でき、コメントや「いいね」を通じて交流が可能。
学習目的で作成し、将来的には React や Flutter Web を用いたフロントエンド実装へ拡張予定。

## ■ 開発目的
- Ruby / Ruby on Rails の実践的な学習
- Rails MVC 構造を理解し、設計・実装・拡張まで経験する
- 将来的に API モード化し、フロントエンド（React・Flutter）と連携できる設計を目指す

## ■ 対応プラットフォーム
- 初期：PC Web（Rails Viewベース）
- 将来：モバイル対応（レスポンシブデザイン）＋ フロント別実装（React / Flutter）

## ■ 目指すサービス像
- **Zenn / Qiita / Note** に近い構成を想定
- Markdown形式で記事作成可能
- 画像・動画添付対応（ActiveStorage使用）
- ゲスト閲覧・コメント・いいね可能
