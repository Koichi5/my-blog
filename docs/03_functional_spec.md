# 🧩 機能仕様（MVP）

## 1. 認証・ユーザー管理

- Devise を利用したログイン / ログアウト / サインアップ機能
- メールアドレス + パスワードでの認証
- ロール（role）による権限管理：`general`, `admin`

## 2. 記事投稿

- Markdown 形式で本文を入力可能
- 画像・動画ファイルを添付可能（ActiveStorage）
- **画像・動画のアップロード方法**：
  - 記事作成画面でドラッグ&ドロップによるアップロード
  - ActiveStorage Direct Upload 機能を使用
  - アップロード完了後、自動的に Markdown 形式の URL がエディタに挿入される
    - 画像: `![alt text](画像URL)`
    - 動画: `[動画](動画URL)`
  - 下書き保存時もアップロード済みファイルは保持される
- 投稿ステータス：下書き（draft）/ 公開（published）
- 公開日時を自動記録（`published_at`）

## 3. 記事閲覧

- 記事一覧表示（新着順・人気順）
- 記事詳細ページ
- 記事ごとのコメント一覧、いいね数表示

## 4. コメント

- 記事ごとにコメント投稿・削除可能
- ログインユーザー or ゲストどちらも投稿可能
- スパム対策（後続対応）

## 5. いいね機能

- ログインユーザーまたはゲスト（Cookie 識別）による「いいね」
- 同一記事に対しては重複いいねを防止
- 非同期処理（Ajax 対応を想定）

## 6. 将来的な拡張予定

- タグ機能（記事へのタグ付け）
- 通知機能（コメント・いいね通知）
- プロフィール編集（アイコン・自己紹介）
- API モード対応 → React / Flutter Web フロント実装
